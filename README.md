1. Инструкция по запуску клиентского приложения на Ubuntu/Debian дистрибутиве:

1.1 Установка QT Creator

Выполните в терминале команду:
"sudo apt update && sudo apt install -y qtcreator"

1.2 Запуск приложения
   
Откройте QT Creator, выполнив команду: 
"qtcreator"

Затем нажмите кнопку "Open Project..." в левом верхнем углу, перейдите в директорию проекта и выберите файл "coursework.pro".
Для запуска нажмите сочетание клавиш "Ctrl+R".



2. Инструкция по запуску клиентского приложения на Windows:

2.1 Установка QT Creator

Перейдите на сайт "https://www.qt.io/product/development-tools" и установите QT Creator.

2.2 Запуск приложения
   
Откройте QT Creator
Затем нажмите кнопку "Open Project..." в левом верхнем углу, перейдите в директорию проекта и выберите файл "coursework.pro".
Для запуска нажмите сочетание клавиш "Ctrl+R".
